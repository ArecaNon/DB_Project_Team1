Team1-phase3 - readme.txt(수정)
프로그램을 실행시키게 되면 팀프로젝트 db가 저장된 곳에 연결을 실행하고 초기 메뉴를 출력합니다.
초기메뉴는
1. 회원가입 2. 로그인  3.종료	로 구성되어있습니다.
1번을 선택하게 되면 회원가입 메소드를 실행시켜 
회원가입에 필요한 정보인 이름,생년월일,주민번호 뒷자리, 성별을 입력하게 하고, 
차량등록 여부를 물어본후 동의하면 차번호,차량모델, 장애인 등록 차량 여부를 물어보고 회원가입을 진행합니다.

2번 메뉴를 선택하게 되면 이름과 주민번호뒷자리(7자리)를 입력 받아 DB에 저장되어있는 사람인지 판단하고
다음 메뉴로 넘어가게 됩니다.
로그인에 성공시 일반 사용자와 관리자로 나눠집니다.
일반 사용자는  예약,예약확인, 가게 검색, 할인 정보 검색, 예약 취소, 가게 자리 검색, 자신차량확인 기능을 사용 할 수 있게 됩니다.

-예약메뉴를 선택하게 되면 날짜와 인원수 ,이용하고 싶은 가게를 선택 받고 주차장예약도 같이 할지 입력받아 판단합니다.
예약번호는 1부터 순서대로 DB에 저장되어있기 때문에 총 예약 숫자를 판단후 +1 해준 값을 예약번호로 출력해줍니다.
좌석은 비어있는 가장 작은 숫자의 테이블 번호부터 배정해줍니다.
좌석정보 중 Rno에 예약 번호를 넣고 예약 가능한지에 대한 정보에 예약이 불가능하다라고 업데이트를 해줍니다
차량 역시 조건에 맞는 비어있는(이용가능한,예약가능한) 숫자의 자리부터 배정해줍니다.
주차 공간 좌석도 주차 예약 번호(Rno)에 예약번호를 저장하고 예약 가능한지에 대한 정보에 예약이 불가능하다고 업데이트 해줍니다
예약은 회원마다 하나씩 밖에 하지 못하게 설계하였습니다.
-예약 확인 기능은 예약번호와 좌석, 주차장 예약도 했을 경우 주차장예약번호와 주차장 자리를 출력시켜줍니다.
-가게 검색기능은 조건에 따라 가게ID나 위치 또는 둘다 사용하여 검색하는 기능을 수행합니다.
-할인정보 검색 기능은 검색을 원하는 가게ID와 할인정보 이름을 사용하여 검색하는 기능을 수행합니다.
-예약 취소 기능은 회원이 예약한 정보를 바탕으로 예약한 테이블과 주차자리를 이용가능한것으로 바꾸고 예약정보를 삭제합니다.
-가게 자리 검색기능은 원하는 가게Id를 입력 받아서 그 가게에 테이블이 몇개 남았는제 출력해줍니다.
-자신차량확인 기능은 자신의 차량 정보를 출력해줍니다. 없을시 없다고 출력해줍니다.
-차량 등록 기능은 자신의 차량이 등록되어있지 않을시 등록 시키는 기능입니다.

관리자 메뉴는 특정 rrn을 확인하고 사용하게 됩니다.(관리자 이름 = '조준표', rrn = '1234567')
관리자 메뉴는 삽입기능, 삭제기능, 검색기능, phase2에서 만든 쿼리를 출력하는 기능이있습니다.
-삽입기능은 원하는 테이블을 입력 받아서 attribute 순으로 입력받아서 insert 문을 실행합니다.
-삭제기능은 원하는 테이블을 입력 받아서 primary 값을 입력받아서  delete문을 실행합니다.
-검색기능은 원하는 테이블을 입력 받아서 그 테이블 전체 값을 출력해줍니다.
-phase2에서 만든 쿼리를 출력하는 기능은 말그대로 phase2에서 만든 쿼리를 출력해서 
해당 프로그램이 정상 작동하는지 여부를 판단 할 수 있습니다.
